Inicio rápido
Cómo instalar y ejecutar la aplicación desde cero.
ℹ️ No se requieren librerías externas. Solo Python 3.10 o superior.
💾 Preparación (primera vez)
  1. Abra una terminal en la carpeta raíz del proyecto.
    La carpeta debe contener las subcarpetas src/ y data/.
  2. Migre los datos históricos — solo la primera vez:
    Terminal
    $ python src/migrate.py
    ✓ Si ya existe data/caja_menor.db, puede omitir este paso. Ejecutarlo de nuevo es seguro (no duplica datos).
  3. Ejecute la aplicación:
    Terminal
    $ python src/app.py
  4. Verá el menú principal:
    Caja Menor
    ========================================== SISTEMA DE CAJA MENOR Consecutivo actual: #1212 ========================================== 1. Registrar movimiento 2. Ver movimientos 3. Ver resumen por item 4. Generar liquidacion 5. Administrar catalogos 0. Salir Opcion: _
💡 Consejos
  • En cualquier punto del menú puede escribir X + Enter para cancelar y volver al inicio.
  • Las fechas siempre se ingresan en formato AAAA-MM-DD (ejemplo: 2026-04-11).
  • Para las fechas, presionar Enter sin escribir nada usa la fecha de hoy.
  • Los montos se ingresan en pesos sin puntos ni comas (ejemplo: 350000).
  • La base de datos se guarda automáticamente en data/caja_menor.db.
1. Registrar movimiento
Ingresa un nuevo movimiento: asignación, gasto, reembolso o ajuste.
🔧 Generador de secuencia

Seleccione los valores y haga clic en Generar. La herramienta le muestra exactamente qué escribir en cada paso de la terminal.

ℹ️ Guía de tipos de movimiento
TipoCuándo usarlo
ASIGNACIONSe le entrega dinero al trabajador para gastos de la obra.
REEMBOLSOSe le devuelve dinero por gastos comprobados con soporte.
GASTOEl trabajador reporta un gasto realizado con el dinero asignado.
AJUSTECorrección contable de un monto registrado previamente.
